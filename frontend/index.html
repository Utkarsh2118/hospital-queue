Everything the queue actually needs

Real-time updates

Every check-in, call, and completion pushes instantly to patients, doctors, and the waiting-room display — no refreshing.

Emergency triage

Emergency check-ins jump the queue automatically, so urgent cases are never stuck behind routine visits.

QR token tracking

Every ticket includes a QR code linking straight to a live tracking page — patients can watch their status from their own phone.

Role-based dashboards

Doctors see their own department's queue. Admins see every department, every doctor, and every token in one place.

How it works

Three steps, start to finish

  1. 1

    Check in at the kiosk

    Pick a department, add basic details, and flag it as an emergency if needed. A token prints instantly.

  2. 2

    Watch it move, live

    Scan the QR code or open the tracking page to see queue position and estimated wait update in real time.

  3. 3

    Get called by the doctor

    When it's time, the token is called on the waiting-room display and the doctor's dashboard — no names shouted across the room.

Security & access

Staff access is locked down. Patients never need an account.

Patients check in without creating an account or storing a password, which keeps the system's attack surface small. Every doctor and admin account is protected the same way:

  • Passwords are hashed, never stored in plain text
  • Sessions are verified on every request, not just at login
  • Doctors can only see and act on their own department's queue
  • Admins can deactivate a staff account instantly if access needs to be revoked
